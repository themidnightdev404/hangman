# Виселица

Консольная игра "Виселица" на языке Go.

## Как запустить

### Требования
- Go 1.20+
- Git

### Установка и запуск
```
git clone https://github.com/themidnightdev404/hangman.git
cd hangman
go run main.go
```

## Как играть
- Введите 1 чтобы начать новую игру
- Вводите по одной букве русского алфавита
- У вас 6 попыток
- Угадайте слово раньше чем виселица будет нарисована
